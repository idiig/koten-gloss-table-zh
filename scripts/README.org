* =generate_available_glosses=

  in json document.

  The output is as follows:

        {
    	"abbreviation": "1SG",
    	"meaning-en": "",
    	"meaning-zh": "",
    	"meaning-ja": ""
        },
        {
    	"abbreviation": "CJT",
    	"meaning-en": "",
    	"meaning-zh": "",
    	"meaning-ja": ""
        },
        ...
    ]
  #+end_src
